## Run it

Use the following steps to run this Project locally:

- Create a directory

- Open the terminal

- Run this command:
`git clone git@github.com:Juanse7793/Catalog-in-Ruby.git`

- Enter in to the folder:
`cd Catalog-in-Ruby`

- Open the folder with a code editor (VS Code preferred)
